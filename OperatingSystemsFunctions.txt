Memory Management allocates memory to different programs.
Assigning different Blocks to optimize overall system performance.
Memory Management is handled at various levels.
It allows for multitasking.
It Prevents Memory-Leaks which would crash the system by keeping issues isolated to specific applications.

File Management helps organize where files are stored.
It assists in finding those files when searched for them.
It Created Indexes that aid in that process
It logs when files were accessed, by whom, and is an important part of looking for breaches in security.
